Introduction and intermediate principles of HTML


My studies first started with learning the basic principles of HTML, including what HTML actually is,
several different kind of elements like heading elements, paragraph elements, list elements and so on.
This then went on to nesting and indentation, file paths on one's computer, the HTML boilerplate and webpages.

All of this knowledge was what I used to create my first project - a personal CV/portfolio page, which can be seen to the right.

This then led to my introduction to CSS, including what CSS is and why it is needed, what seperation of concerns is,
the different ways to apply CSS to ones webpages (internally, externally, etc.), CSS selector, etc
This was followed by the learning of applying different properties using CSS (e.g. colour, fonts properties), inspecting
CSS and the CSS box model. I also learned how to use flexbox in tandom with CSS for easier building on website structure
as well as basics on web design (colour theory, typography, UX/UI design).

I used all of this information and knowledge to create this very website, for the purpose of showcasing my learned and applied
knowledge regarding CSS and HTML up until this point. An image of the homepage can be seen to the left. Use of intermediate and advanced
CSS and HTML concepts were used in the creation of this website, as well as use of flexbox and web design principles.
